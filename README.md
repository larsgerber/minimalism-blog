# NodeJS frontend for blog.larsgerber.ch

## Running the Project

``` none
npm install
npm run gulp
npm run dev
```

``` none
npm install
npm run build
npm run test
node app.js
```

🐳 Docker

``` none
docker-compose build --no-cache
docker-compose up -d --force-recreate --remove-orphans
docker image prune --all --force
```
